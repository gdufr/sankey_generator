Welcome to the E2E Relationship Tracker, developed by the Rebellion Platform architecture team. Simply select an L2 (or multiple L2s) to see the relationships between the layers. There will be four columns generated:
  • Rebellion Screen
  • Rebellion Facade API
  • Core API
  • Ericsson WSIL
You can click on the colored rectangle immediately next to a label to see the direct upstream and downstream relationships for a given node. The source data is the E2E Mapping spreadsheet. Note that an "NA" value implies either a missing data field in the spreadsheet or that there is no Ericsson service (as in Store operations).
